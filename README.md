# Gokula-Health 🐄📱

Gokula-Health is an Android application designed to help dairy farmers efficiently manage cattle health, milk production, breeding cycles, and vaccination schedules. The app acts as a digital health record system for livestock, improving productivity and reducing financial losses due to missed care or irregular tracking.

---

## 🚀 Features

- 🐄 Cattle Profile Management (Ear Tag ID, Photo, Details)
- 🥛 Daily Milk Yield Tracking (Morning & Evening entries)
- 📊 Milk Production Graphs using MPAndroidChart
- 🔔 Vaccination & Health Reminders using AlarmManager
- 📅 Heat Cycle & Breeding Tracking
- 💾 Offline Data Storage using Room Database

---

## 🛠️ Tech Stack

- Kotlin (Android Development)
- Room Database
- AlarmManager
- MPAndroidChart
- Android Studio

---

## 🎯 Objective

To improve dairy farm productivity by providing a smart, offline, and easy-to-use livestock management system that helps farmers track cattle health, milk yield trends, and vaccination schedules effectively.

---

## 🌱 Impact

- Increased dairy productivity
- Reduced missed vaccinations
- Better livestock health monitoring
- Supports rural digital transformation
